# Proyecto: Sistema de Gestión de Citas - Centro Integral Alerce

## 📖 Descripción General
El proyecto **Centro Integral Alerce** es una aplicación que permite gestionar las citas y actividades agendadas en el centro integral, facilitando el control, seguimiento y administración de los servicios que se ofrecen.  
El sistema busca mejorar la eficiencia en la gestión interna, optimizando los procesos de registro, asignación y control de citas, con distintos roles de usuario y acceso a información relevante.

## 🎯 Objetivos

### Objetivo General
Desarrollar una aplicación que permita gestionar de manera eficiente las citas y actividades del Centro Integral Alerce.

### Objetivos Específicos
- Permitir el registro, modificación y eliminación de citas.
- Controlar y supervisar los archivos asociados a cada cita.
- Gestionar usuarios con distintos niveles de acceso (administrador, usuario y revisor).
- Diseñar una interfaz clara, accesible y adaptable a dispositivos web.
- Mantener la información almacenada de forma segura en una base de datos.

## ⚙️ Funcionalidades Principales
- Registro y gestión de citas.
- Control de archivos (subir, editar, eliminar).
- Gestión de usuarios con roles definidos:
  - **Administrador**: Acceso total al sistema.
  - **Usuario**: Puede registrar y ver citas.
  - **Revisor**: Puede revisar, subir, editar o eliminar archivos.
- Generación de reportes o listados de actividades.
- Interfaz moderna y responsiva.
